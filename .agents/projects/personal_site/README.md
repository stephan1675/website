# Project: Personal Portfolio Site

## Overview
This project represents the public-facing landing page and mechatronics portfolio for Stephan Jeremias. It is designed to impress recruiters and mechatronics professionals with clean glassmorphic UI and provide structured information for search engines and AI agents.

## Folder Structure & Contents
- Public website files are located at the root of the repository (`index.html`, `js/`, `css/`).
- Place design guides, visual assets, and content drafts (like LinkedIn posts or project descriptions) here.

## Roadmap & Features
1. **AI Optimization**: Standardize Person schema and maintain `/llms.txt` for AI scrapers.
2. **Project Showcases**: Build separate detailed sections for the wheelchair control study, RC Car, and C++ games.
3. **Interactive Demos**: Implement retro terminal or visual mechatronics simulators.
